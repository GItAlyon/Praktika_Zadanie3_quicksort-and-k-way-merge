# QuickSort + K-Way Merge (TupleSort)

Реализация алгоритмов сортировки на C++: быстрая сортировка (QuickSort), сортировка слиянием (MergeSort) и внешняя сортировка с K-путевым слиянием (K-Way Merge / External Sort).

## 📖 Описание

Проект реализует три основных алгоритма сортировки:

1. **Быстрая сортировка (QuickSort)** — один из самых быстрых алгоритмов сортировки, работающий по принципу "разделяй и властвуй". Выбирает опорный элемент (средний) и разбивает массив на две части.

2. **Сортировка слиянием (MergeSort)** — устойчивый алгоритм сортировки, гарантирующий время O(n log n). Используется для демонстрации принципов параллельной и внешней сортировки.

3. **Внешняя сортировка (External Sort / K-Way Merge)** — алгоритм для сортировки данных, не помещающихся в оперативную память. Разбивает данные на чанки, сортирует их, а затем выполняет K-путевое слияние.

Также реализована **параллельная версия QuickSort** с использованием потоков для ускорения сортировки больших массивов.


## 🚀 Алгоритмы

### Быстрая сортировка (QuickSort)

```cpp
Sorter<int>::quickSort(arr);
```

*Характеристики:*

- Среднее время: O(n log n)

- Худшее время: O(n²)

- Дополнительная память: O(log n)

- Выбор опорного элемента: средний

### Сортировка слиянием (MergeSort)
```cpp
Sorter<int>::mergeSort(arr);
```

*Характеристики:*

- Время: O(n log n) всегда

- Дополнительная память: O(n)

- Устойчивая сортировка

### Внешняя сортировка (External Sort / K-Way Merge)
```cpp
Sorter<int>::externalSort("input.bin", "output.bin", 100); // 100 - лимит памяти в МБ
```

*Характеристики:*

- Подходит для данных, не помещающихся в RAM

- Разбиение на чанки + K-путевое слияние

- Настраиваемый лимит памяти

### Параллельная сортировка (Parallel QuickSort)
```cpp
Sorter<int>::parallelQuickSort(arr, 4); // 4 - количество потоков
```

*Характеристики:*

- Использует std::thread

- Оптимален для массивов > 10000 элементов

- Автоматическое слияние результатов
